# 基础项目框架

## 项目概述

这是一个基于FastAPI + Vue3的前后端分离项目框架，提供快速原型开发和热重载功能。

## 项目结构

```
├── backend/           # FastAPI后端服务
├── frontend/          # Vue3前端应用
├── docs/             # 项目文档
├── specs/            # 功能规格文档
└── README.md         # 项目说明
```
